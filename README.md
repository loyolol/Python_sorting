Лабораторная работа: Алгоритмы сортировки
📋 Описание проекта
Данный проект представляет собой реализацию пяти различных алгоритмов сортировки на языке Python. Работа выполнена в рамках учебного задания и включает как базовые алгоритмы, так и их оптимизированные версии, а также практическое применение сортировки для обработки данных.

📁 Структура проекта
Задание №1: Пузырьковая сортировка (test1.py)
Реализация классического алгоритма пузырьковой сортировки с подсчетом метрик.

Функциональность:

Сортировка массива по возрастанию

Вывод исходного и отсортированного массива

Подсчет количества сравнений и обменов

Пример:

text
Исходный массив: [5, 1, 4, 2, 8]
Отсортированный массив: [1, 2, 4, 5, 8]
Количество сравнений: 10
Количество обменов: 4
Задание №2: Сравнение алгоритмов (test2.py)
Реализация и сравнение двух алгоритмов сортировки.

Алгоритмы:

Сортировка выбором (Selection Sort)

Сортировка вставками (Insertion Sort)

Особенности:

Измерение времени выполнения (в миллисекундах)

Сравнение производительности на одинаковых данных

Пример:

text
Сортировка выбором: [1, 3, 6, 7, 9] (время: 0.0000 мс)
Сортировка вставками: [1, 3, 6, 7, 9] (время: 0.0000 мс)
Задание №3: Оптимизированный пузырек (test3.py)
Улучшенная версия пузырьковой сортировки с ранним завершением.

Оптимизация:

Добавлен флаг swapped для проверки отсортированности

Автоматическое завершение при отсутствии обменов

Подсчет количества проходов

Пример:

text
Массив уже отсортирован.
Количество проходов: 1
Задание №4: Быстрая сортировка (test4.py)
Рекурсивная реализация алгоритма Quick Sort.

Особенности:

Рекурсивный подход

Выбор среднего элемента в качестве опорного (pivot)

Разделение на подмассивы (меньше, равные, больше опорного)

Пример:

text
Исходный массив: [10, 7, 8, 9, 1, 5]
Отсортированный массив: [1, 5, 7, 8, 9, 10]
Задание №5: Реальная задача (test5.py)
Применение сортировки для обработки списка студентов.

Условия сортировки:

По среднему баллу (по убыванию)

При равных баллах — по фамилии (по алфавиту)

Вывод топ-5 студентов

Пример:

text
Топ студентов:
Петров Иван 4.8
Сидоров Алексей 4.8
Иванов Пётр 4.5
Смирнова Анна 4.2

📊 Анализ алгоритмов
Сравнение сложности
Алгоритм	Лучший случай	Средний случай	Худший случай	Память
Пузырьковая	O(n)	O(n²)	O(n²)	O(1)
Пузырьковая (оптим.)	O(n)	O(n²)	O(n²)	O(1)
Выбором	O(n²)	O(n²)	O(n²)	O(1)
Вставками	O(n)	O(n²)	O(n²)	O(1)
Быстрая	O(n log n)	O(n log n)	O(n²)	O(log n)
Ключевые выводы
Пузырьковая сортировка — проста для понимания, но неэффективна на больших данных

Оптимизированная версия показывает хорошие результаты на почти отсортированных массивах

Сортировка вставками эффективна для небольших массивов

Быстрая сортировка — лучший выбор для случайных данных большого объема

Встроенная функция sorted() в Python оптимизирована и использует Timsort

🎯 Цель работы
Закрепление теоретических знаний об алгоритмах сортировки и получение практических навыков их реализации на языке Python. Работа демонстрирует понимание:

Различных подходов к сортировке данных

Важности оптимизации алгоритмов

Применения сортировки для решения практических задач

Анализа временной сложности алгоритмов

📝 Примечания
Все алгоритмы реализованы с нуля для учебных целей

Для замера времени используется модуль time
