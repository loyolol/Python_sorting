# Лабораторная работа: Алгоритмы сортировки

Данный проект содержит реализацию пяти классических алгоритмов сортировки на языке Python.  
Каждое задание оформлено в отдельном файле и демонстрирует работу соответствующего алгоритма с подсчётом метрик (сравнения, обмены, время выполнения) или оптимизациями.

## Структура проекта

- `test1.py` — пузырьковая сортировка (Bubble Sort) с подсчётом сравнений и обменов.
- `test2.py` — сравнение сортировки выбором (Selection Sort) и сортировки вставками (Insertion Sort) по времени выполнения.
- `test3.py` — оптимизированная пузырьковая сортировка с ранним завершением.
- `test4.py` — быстрая сортировка (Quick Sort).
- `test5.py` — практическое применение: сортировка списка студентов по среднему баллу и фамилии.

---

## Задание 1. Пузырьковая сортировка

**Файл:** `test1.py`

Реализована сортировка пузырьком для массива целых чисел. Программа выводит исходный массив, отсортированный массив, количество сравнений и количество обменов.

**Пример вывода:**
Задание 1
Исходный массив: [5, 1, 4, 2, 8]
Отсортированный массив: [1, 2, 4, 5, 8]
Количество сравнений: 10
Количество обменов: 4

## Задание 2. Сравнение алгоритмов

**Файл:** `test2.py`

Реализованы сортировка выбором (selection sort) и сортировка вставками (insertion sort). Для каждого алгоритма измеряется время выполнения в миллисекундах.

**Пример вывода:**
Задание 2
Сортировка выбором: [1, 3, 6, 7, 9] (время: 0.0038 мс)
Сортировка вставками: [1, 3, 6, 7, 9] (время: 0.0029 мс)

## Задание 3. Оптимизированный пузырёк

**Файл:** `test3.py`

Пузырьковая сортировка улучшена за счёт флага `swapped`: если за проход не было ни одного обмена, алгоритм завершается досрочно. Выводится количество выполненных проходов и сообщение, если массив уже был отсортирован.

**Пример вывода:**
Задание 3
Массив уже отсортирован.
Количество проходов: 1

## Задание 4. Быстрая сортировка

**Файл:** `test4.py`

Реализована быстрая сортировка (Quick Sort) с выбором опорного элемента из середины массива. Алгоритм рекурсивно сортирует подмассивы и объединяет их.

**Пример вывода:**
Задание 4
Исходный массив: [10, 7, 8, 9, 1, 5]
Отсортированный массив: [1, 5, 7, 8, 9, 10]

## Задание 5. Сортировка в реальных программах

**Файл:** `test5.py`

Дан список кортежей с фамилией, именем и средним баллом студентов. С помощью встроенной функции `sorted` выполняется многоуровневая сортировка: сначала по убыванию балла, затем по фамилии (по возрастанию). Выводится топ-5 студентов.

**Пример вывода:**
Задание 5
Топ студентов:
Петров Иван 4.8
Сидоров Алексей 4.8
Иванов Пётр 4.5
Смирнова Анна 4.2
